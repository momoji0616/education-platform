# rag
本项目拟开发一套“融合 RAG（Retrieval-Augmented Generation，检索增强生成）的智能教育平台”。平台以高校学生为主要服务对象，围绕学生基础信息、学习成绩与学习行为数据，构建统一的数据管理与智能分析环境。在现有学生信息导入、数据库查询问答和基础成绩预测功能的基础上，引入 RAG 技术，将结构化数据库数据与非结构化教育文本（如课程说明、学习建议模板、教学管理制度等）相结合，实现更准确、更具上下文依据的智能问答与学习支持服务。
